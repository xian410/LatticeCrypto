# 项目介绍

格密码军科委项目前端汇总版
技术栈：
前端：electron+vue，后端：nodejs
启动流程：

1. 安装 nodejs，npm
2. 进入项目，npm install 安装 node modules, cd backend 安装 node modules
3. 启动 node js
4. npm run start 打开 electron

## 项目目录结构

eletctron 前端 -> nodejs 后端 -> 读写 txt 文件
|-> 算法可执行文件
